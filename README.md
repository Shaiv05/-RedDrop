## 🌟 What is RedDrop?

<img align="right" width="360" src="https://raw.githubusercontent.com/andreasbm/readme/master/assets/lines/rainbow.png" />

<table>
<tr>
<td>

**RedDrop** is a full-stack web application built on the **MERN Stack** that revolutionizes the blood donation experience. It creates a seamless digital ecosystem connecting three critical groups:

| Role | Description |
|------|-------------|
| ❤️ **Blood Donors** | Register, schedule donations & track history |
| 🏥 **Blood Banks** | Manage inventory, donors & requests digitally |
| 🚑 **Emergency Recipients** | Submit urgent requests & find nearest donors |

Built with speed, security, and compassion — RedDrop is designed to save lives through technology.

</td>
</tr>
</table>

## ⚙️ Getting Started

### 📋 Prerequisites

Make sure you have the following installed:

![Node](https://img.shields.io/badge/Node.js-v18+-339933?style=flat-square&logo=nodedotjs)
![npm](https://img.shields.io/badge/npm-v9+-CB3837?style=flat-square&logo=npm)
![MongoDB](https://img.shields.io/badge/MongoDB-Atlas_or_Local-4EA94B?style=flat-square&logo=mongodb)

---

### 🚀 Installation & Setup

**1️⃣ Clone the Repository**

```bash
git clone https://github.com/YOUR_USERNAME/RedDrop.git
cd RedDrop
```

**2️⃣ Configure Environment Variables**

```bash
# Create .env in /backend
cp .env.example backend/.env

# Edit and fill in:
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_super_secret_key
PORT=5000
```

**3️⃣ Install Dependencies**

```bash
# 📦 Install Frontend
cd frontend && npm install

# 📦 Install Backend
cd ../backend && npm install
```

**4️⃣ Run the Application**

```bash
# 🎨 Start Frontend (Vite Dev Server)
cd frontend && npm run dev

# ⚙️ Start Backend (Express API)
cd backend && npm start
```

**5️⃣ Open in Browser**

```
Frontend  →  http://localhost:5173
Backend   →  http://localhost:5000
```

## 🗺️ API Endpoints Overview

```
🔐 AUTH ROUTES          /api/auth
   POST  /register       ← Register new user
   POST  /login          ← Login & get JWT token
   GET   /me             ← Get current user profile

🩸 DONOR ROUTES         /api/donors
   GET   /               ← List all donors
   POST  /register       ← Register as donor
   GET   /:id            ← Get donor by ID
   PUT   /:id            ← Update donor profile

🚨 REQUESTS ROUTES      /api/requests
   POST  /               ← Create emergency blood request
   GET   /               ← View all active requests
   PUT   /:id/fulfill    ← Mark request as fulfilled

🏥 BANK ROUTES          /api/banks
   GET   /               ← List blood banks
   GET   /:id/inventory  ← Get bank inventory
```
