🚀 About RedDrop

🩸 RedDrop is a modern full-stack blood donation platform built using the MERN Stack.

It helps connect:

* ❤️ Blood Donors
* 🏥 Blood Banks
* 🚑 Emergency Recipients

The platform streamlines blood requests, donor management, authentication, and emergency response through a clean and responsive user experience.

⸻

✨ Features

✅ Secure Authentication
✅ Donor Registration System
✅ Emergency Blood Requests
✅ Modern Responsive UI
✅ Fast MERN Stack Architecture
✅ Real-Time Data Handling
✅ Organized Dashboard Structure
✅ Mobile Friendly Design
